## block-y

[Live link](block-y.herokuapp.com)

1. clone repo   
```
git clone https://github.com/tbuchannan/block-y.git
```

2. cd into directory   
```
cd block-y
```

3. install dependencies using npm or yarn   
```
yarn install or npm install
```

4. run
```
yarn dev
```

  or
```
npm run dev
```
