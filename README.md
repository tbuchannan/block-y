## BLOCKCHAIN Code Challenge

### To run coding challenge

1. clone repo   
```
git clone https://github.com/tbuchannan/blockchain-cc.git
```

2. cd into directory   
```
cd blockchain-cc
```

3. install dependencies using npm or yarn   
```
yarn start
```
